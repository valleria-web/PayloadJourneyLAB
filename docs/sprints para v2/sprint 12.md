# Requisitos editoriais

* Preserve o idioma e o tom editorial actual do site.
* Não transforme a página numa narrativa autobiográfica excessivamente longa.
* Mantenha a fundadora presente, mas preserve o LAB como sujeito institucional central.
* Não declare o campo como oficialmente reconhecido.
* Não declare que o piloto já foi concluído.
* Não declare que outras pessoas já reproduziram o protocolo, a menos que exista evidência no repositório.
* Não utilize expressões como “primeiro laboratório do mundo”, “único método” ou outras afirmações não verificáveis.
* Diferencie sempre visão, objectivo, trabalho em andamento e resultado comprovado.
* Preserve o princípio já existente de apresentar somente factos confirmados.
* Não invente números, parceiros, instituições, datas exactas ou resultados.
* Use “horizonte inicial de seis meses”, sem criar uma data final que não esteja definida no código ou conteúdo existente.
